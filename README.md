# Jornada Viagens

Landing page responsiva para uma agência de viagens fictícia, desenvolvida com HTML5 e CSS3. O projeto foi criado para praticar estrutura semântica, responsividade e organização de estilos utilizando CSS modular.

🔗 Demo

## Tecnologias

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
* Variáveis CSS

## Funcionalidades

* Layout responsivo para mobile, tablet e desktop.
* Menu mobile sem JavaScript.
* Seção de ofertas com cards de viagem.
* Busca por categorias.
* Destinos populares.
* Área de condições de pagamento com CSS Grid.
* Depoimentos de clientes.
* Banner pré-footer e rodapé responsivo.

## Como Executar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` no navegador.

Opcionalmente, utilize a extensão Live Server no VS Code.
